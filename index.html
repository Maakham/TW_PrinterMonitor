Introduction

Tailwind CSS เป็น Framework แบบ Utility-first ที่แตกต่างจาก Bootstrap อย่างสิ้นเชิง

หลักการ: แทนที่จะใช้ Component สำเร็จรูป (เช่น btn), Tailwind ให้คุณใช้คลาสเล็กๆ (Utility) ประกอบกันเป็นรูปร่างที่คุณต้องการ

ทำไมต้องใช้ Tailwind?

  • ไม่อ้วน: เมื่อนำขึ้น Production จะตัด CSS ที่ไม่ใช้ออก ไฟล์เล็กมาก
  • ไม่ต้องแก้ชื่อ Class: ไม่ต้องมานั่งคิดชื่อ class อย่าง sidebar-inner-wrapper อีกต่อไป
  • ยืดหยุ่น: ปรับแต่งได้ทุกอย่าง ไม่ติดกับหน้าตาเดิมๆ ของ Framework

Installation

วิธีที่ง่ายที่สุดในการเริ่มต้นเรียนรู้ คือการใช้ CDN (Play CDN)

ใช้งานผ่าน CDN

เพียงแค่แปะ Script นี้ลงใน <head> ของไฟล์ HTML

<script src="https://cdn.tailwindcss.com"></script>

หมายเหตุ: CDN เหมาะสำหรับการพัฒนาและการเรียนรู้เท่านั้น สำหรับ Production จริง แนะนำให้ติดตั้งผ่าน npm (Node.js)

Spacing

การจัดการ Margin และ Padding คือหัวใจหลักของความสวยงาม

Code Format

  • m-{size} = margin รอบด้าน
  • mx-{size} = margin ซ้าย-ขวา
  • my-{size} = margin บน-ล่าง
  • mt-{size} = margin บน
  • p-{size} = padding รอบด้าน

Scale (ขนาด)

  • 1 = 0.25rem (4px)
  • 2 = 0.5rem (8px)
  • 4 = 1rem (16px) **ใช้บ่อย**
  • 8 = 2rem (32px)
